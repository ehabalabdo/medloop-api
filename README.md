# MedLoop API

- Auth (JWT)
- Appointments
- Calendar
- Reports
- RBAC

## Quick Start

1. npm install
2. npm start

## Endpoints
- /auth/login
- /auth/refresh
- /appointments (CRUD)
- /appointments/today, /week, /day
- /reports/doctor-load, /cancellations, /peak-hours

## Features
- JWT authentication
- Role-based access
- Calendar queries
- Audit logs
- Validation (Zod)
